# How to use

If you want to test application in the local, run from cmd:

```bash
npm run debug
```

Also, the application is available on AWS Elastic Beanstalk:
```bash
http://nodejsrest-env.eba-zdqqhzkd.us-east-2.elasticbeanstalk.com/
```
